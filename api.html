element_id¶

str – A UDID used to uniquely identify an element.

clear()[source]¶

The Element Clear command scrolls into view a submittable element excluding buttons or editable element, and then attempts to clear its value, checkedness, or text content.

click()[source]¶

The Element Click command scrolls into view the element and then attempts to click the centre of the visible area of the first element of the DOMRect sequence. In case the element is not displayed, an element not visible error is returned.

driver¶

Internal reference to the WebDriver instance.
